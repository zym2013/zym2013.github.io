【插件】Amazing Luogu

zhangyimin123452025-10-092025-10-09

看到了许多洛谷插件，于是自己也写了个。

插件文档：https://zym2013.github.io/amldocs。

一些功能

私信渲染

自动渲染私信，支持 Markdown，支持 Katex，但不支持代码块渲染。

请注意链接可自动识别，但图片不会（包括 .png 结尾的），需要：

私信里不常用代码块，所以不渲染代码。

主界面

包含每日古诗、历史上的今天、公告、设置、搜索。

题目颜色显示

找到题号并修改其颜色为它的难度。

复制 Markdown

支持专栏、题单、比赛、讨论界面。

其中专栏、题单、比赛界面 Ctrl + Alt + C 复制内容。

还写了暂时全网唯一的讨论区复制源码，主内容以及每个回复都设有复制按钮。

跳题器

双击题号即可跳转至对应洛谷题目。

其他

备忘录、首页犇犇自动展开与屏蔽用户、比赛报名按钮解锁器、长代码自动套折叠框、自动跳转保存站、首页自动签到、删除私信等。

获取插件

安装好篡改猴后打开此网址：

http://gh.halonice.com/https://raw.githubusercontent.com/SnakeBloc-zym/Amazing-Luogu/refs/heads/main/index.user.js
